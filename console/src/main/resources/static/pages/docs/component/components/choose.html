分支选择步骤

分支选择用来做多分支判断，只有满足指定条件，才会继续执行后续步骤

{
  "id": string,
  type:"choose",
  children:[
   {
     when:MevlScript,// 判断条件，满足条件才会执行children步骤
     children:Array<Step> // when匹配后执行此步骤
   },
   {
     when:MevlScript,

     children:Array<Step> // 匹配后执行此步骤
   },
   { // 默认执行的逻辑

     children:Array<Step> 
   }
  ]
}

choose的多个分支间的执行逻辑是互斥的，即先判断第一个条件，条件满足后停止后续判断，否则会判断后续的条件，直至所有条件都判断完成。

when表达式

when表达式用来做判断，必须返回boolean类型，可以用workflow、steps2个变量。关于mvel脚本，详细参考脚本配置 举例说明：

workflow.input.params.id==1;
